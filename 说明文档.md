影像色调大师 (Color Mimicry Master)

这是一个基于 Web 技术的专业级摄影仿色与调色工具。它可以分析参考图片的色彩分布，并将其光影氛围精准迁移到你的照片上，同时支持导出为标准的 .cube LUT 文件。

🚀 核心功能

色彩迁移：基于 LAB 空间统计算法，一键复制摄影大片的色彩风格。

专业修图面板：支持曝光、对比度、色温、色调、饱和度及胶片颗粒调节。

实时预览：支持原图与处理效果的瞬间对比。

导出 LUT：生成 32 阶 3D LUT 文件，可直接导入 Photoshop、Premiere、DaVinci Resolve。

🛠 使用技术

HTML5 Canvas (图像处理核心)

Tailwind CSS (响应式界面)

Lucide Icons (图标系统)

LAB Color Space Conversion (色彩科学算法)

📦 快速部署

本项目为单文件架构，无需服务器后端。直接将 index.html 部署于 GitHub Pages、Vercel 或 Netlify 即可使用。
